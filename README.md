# Panopto ICP Scoring Configuration Tool

Interactive tool for configuring and visualizing ICP scoring criteria across Manufacturing and Non-Banking Finance segments.

Built by [Locality AI](https://locality.ai) for Panopto.

## Features

- Configure FIT, SIGNALS, and TIMING scoring weights
- Filter and preview universe data by sub-industry
- Real-time score distribution visualization
- Save/share configurations via URL hash
- Export configurations as JSON

## Deployment

Deployed automatically to GitHub Pages via GitHub Actions on push to `main`.
